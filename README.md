# 词对 PK 🎯

双人英中单词配对对战游戏 · 单文件 HTML · 教室一体机专用

## 功能

- 👫 **双人模式** — 同屏分边竞速，先清空 8 对者胜
- 🧑 **单人挑战** — 全屏网格，刷自己最快记录
- 📖 **词表** — 内置人教版高中英语必修 1~选必 4，共 2331 词
- 🔊 **TTS 发音** — 配对成功自动朗读英文（可开关）
- 🏆 **排行榜** — LocalStorage 存储，支持导入/导出迁移
- 📱 **移动端适配** — 手机自动上下堆叠布局
- 🎨 **教室友好** — 暖白质感 + 樱花粉配色
- 👆 **多点触控** — 左右区域独立响应
- 📦 **单文件** — 双击即开，零安装

## 玩法

1. 双击 `index.html` 打开
2. 选词表 → **开始 PK**（双人）或 **单人挑战**
3. 3-2-1 倒计时后，点击英文和对应中文配对消除
4. 先清空 8 对者胜（双人）/ 记录最快用时（单人）

## 技术栈

Vue 3 · Noto Sans SC · speechSynthesis · LocalStorage · 纯前端
